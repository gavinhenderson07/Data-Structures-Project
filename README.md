A C++ implementation of core data structures including binary search, hash tables, priority queues, and linked lists. This project demonstrates clean, efficient implementations suitable for technical interviews and coding assessments.

## Overview

This repository contains self-contained implementations of fundamental data structures commonly used in software development. Each module is designed to be clear and easy to understand, making it ideal for reviewing algorithmic concepts or as a reference during technical interviews.

## Project Structure

- **binarysearch.cpp / .h** - Binary search implementation with utilities
- **hash_table.cpp / .h** - Hash table implementation with collision handling
- **priorityqueue.cpp / .h** - Priority queue using heap-based structure
- **LinkedListNode.cpp / .h** - Linked list node and utilities
- **charmatrix.cpp / .h** - Character matrix operations and helpers
- **config.cpp / .h** - Configuration utilities
- **main.cpp** - Driver program with example usage
- **run.bash** - Build and execution script
